Outstanding Webs

For this task, we had to select one web project per awarder from among the nominees and honored ones, and write a brief explanation for each including an analysis and reasons for considering it an outstanding web design.

From a coding bootcamp to a Parisian hotel, I found four amazing websites.

Analysis 1: NOD Coding

URL of the nominees: https://www.awwwards.com/websites/nominees/

URL of the analyzed website: https://nodcoding.com/

I believe this web design is outstanding for several reasons.

  • Color palette: The color palette creates their brand identity, it is harmonious and I feel at ease with pastel colors.
  • Imagery: Regarding imagery and graphics, what caught my attention to begin with is the animation in the homepage and that with the mouse of the computer, you can move it as well if you hover on the animation. I think this is innovative and interactive. I also liked the way they wrote “Coding” with shapes rather than with letters.
  • User experience: It is easy for users to find what they're looking for and from the very first glance at the website it is understandable what it is about which is a bootcamp for coding.
  • It is a responsive website and when the viewport gets smaller, the menu turns into a hamburger menu. Finally, when you hover over the menu (not the hamburger one), there's an animation which I've never seen before and I would love to incorporate it into my web even though I have no idea how to do it.

Analysis 2: Bowery Farming

URL of the nominees: https://winners.webbyawards.com/winners/websites-and-mobile-sites/general-desktop-mobile-sites/food-drink

URL of the analyzed website: https://bowery.co/

This website is innovative, trendy, and visually appealing.

  • Animations: From the start, there's an animation that leads up to the homepage which I've never seen before so I found it surprising in a good way. Throughout the page, there are animations (that are interactive), good quality photographs and illustrations which makes it more visually appealing, especially since the website is about food but I will say it took me a few minutes to understand what the website is about.
  • Typography: They use several fonts which go well with each other and they use different weights; from bold to thinner ones. The one I liked the most is the bold one as it caught my attention multiple times.
  • Placement and menu: I like how their logo is placed in the middle of the header and how when the viewport gets smaller, the menu turns into a hamburger menu (which is trendy).
  • Menu: While scrolling, the menu is not fixed. Instead, it turns into a button which I don't think I've seen before so I found it innovative. Overall, I liked the user experience because it was fun, creative and it invited you to keep learning about the website.

Analysis 3: Firefly Partners

URL of the nominees: https://thefwa.com/awards/page/3

URL of the analyzed website: https://fireflypartners.com/

The website features strong branding through its color palette and carousel elements.

  • Color palette: The first thing that caught my attention was the color palette since it includes a saturated orange, yellow and purple. I believe these colors work very well.
  • Photographs: What I also liked are the photographs they are using; they help transmit their objective very well and they function as examples of their work.
  • Carousels: Moreover, throughout the site there are three carousels of information (which I don't see very often in web design) and I believe it is a good way of making the page more interactive or making the user interested in the information you are providing.
  • Typography: Font pairing is also well done in this page and I believe it keeps the website interesting and visually appealing. I think the only thing I disliked is that the header doesn't occupy the whole viewport.

Analysis 4: Place Vendome

URL of the nominees: https://www.cssdesignawards.com/wotd-award-winners

URL of the analyzed website: https://www.1-placevendome.com/en/

This is an innovative website that uses horizontal scrolling which creates a unique experience.

  • Scrolling: In Place Vendome, you can scroll from right to left and from top to bottom. I think this is very innovative but for some people it may be hard to use or you may get lost quite easily. What helps in this case is that there is a button called “Room collection” on the left upper corner which allows you to see the rooms in a list.
  • Quotes: As you move around, the homepage, quotes start appearing which is also something I've never seen before. Most of them I found quite random as they're not related to the hotel or to the hospitality environment but still I found them surprising in a good way.
  • Photographs: The website shows the different rooms the hotel has in a unique way, they use good quality photographs but more interestingly, they showcase the photos and animated illustrations like a moodboard. This works great, at least for me, because the user gets to have a glimpse of how it would feel to stay in each room and they get to choose the one they like the most.
  • Parisian feeling: From the showcased photos to typography and color, the web design reflects that Parisian feeling. Overall, it is a classy web design that found a way to be innovative but still elegant and interesting.
